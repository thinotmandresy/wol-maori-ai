## What is this?

<p align="justify">
This is a simple AI script for the Maori civilization in the <a href="https://aoe3wol.com/">Wars of Liberty</a> mod for <a href="https://store.steampowered.com/app/105450/Age_of_Empires_III_Complete_Collection/">Age of Empires&#174; III (2007)</a>. I wrote it from scratch in 2020 as an attempt to have a base for future Polynesian civilizations. I left it unfinished in 2021, and I haven't touched it since. Now I'm making a separate, public repository for it, so that it can be continued by anyone who wants to. I might make some changes to it from time to time, but I don't plan on doing anything major.
</p>

## How do I use it?

### Recommended: use the WoL updater

This AI script is already included in the mod, so you don't need to download it separately. Just use the [WoL updater](https://aoe3wol.com/download) to update WoL to its latest version and you'll get the AI version that was included in the mod at the time of its release. However, you won't get any new changes that were made to the AI until the next WoL update.

### Alternative: download this repository

If you can't wait for the next WoL update, you can download the AI script directly from this repository. This way, you'll get the latest version of the AI script, but you'll have to manually update it every time a new version is released. Here's how to do it:

1. Check the [commit history](https://github.com/thinotmandresy/wol-maori-ai/commits/main) to see if updating the AI script is worth it. Sometimes, the changes are kind of insignificant so you might not want to bother.
2. [Download the repository](https://github.com/thinotmandresy/wol-maori-ai/archive/refs/heads/main.zip).
3. Find the folder where you installed Wars of Liberty. It contains the `age3y.exe` file.
4. Inside that folder, find the `AI3` folder. It should contain a file named `_TeRauparaha.xs`. You might want to make a backup of all the files in that folder before continuing.
5. Open the zip file you downloaded at step 2. It contains a folder named `wol-maori-ai-main`. Open that folder and select all the files and folders inside it.
6. Extract the selected files and folders into the `AI3` folder you found at step 4. It should ask you if you want to replace the existing files. Say yes.
7. Open the game and start a skirmish match against the Maori civilization. The AI should be working.

If you have a hard time following these instructions, please [join WoL's Discord server](https://discord.gg/Jpjm9Ja) and ping me in the `#modding` channel.
